# Raider Spinner 2

A random name picker, group maker, and classroom timer built for Regis Jesuit High School teachers.

**[Launch Raider Spinner 2 →](https://rjedtech.github.io/Raider-Spinner-2/)
## Features

- **Spinner** — Tap the wheel to randomly select a student. Toggle "Remove after picked" to avoid repeats, with a visual tracker showing who's already been called.
- **Group Maker** — Shuffle students into groups of 2, 3, 4, or 5. Copy groups to clipboard for pasting into Teams or Canvas.
- **Timer** — Countdown presets (30s to 5min) or custom entry. Pairs with the spinner — pick a name, start the clock.
- **Saved Class Lists** — Save multiple rosters by name (Period 1, AP Micro, etc.). One tap to load next time.
- **Sound Effects** — Tick sounds during the spin, chime on landing, buzz when the timer ends. Mute button for testing days.
- **Fullscreen Mode** — Hides everything except the tool for projecting on the board.

## Privacy

- No accounts, no login
- No ads, no cookies, no tracking
- No data leaves the browser — everything runs locally
- Saved class lists are stored in your browser only (not on any server)
- Nothing is transmitted anywhere, ever

## How to Use

1. Visit the link above and bookmark it
2. Paste student names (one per line or comma-separated)
3. Click **Load Names** — optionally click **Save as Class** to save the roster for next time
4. Tap the wheel to spin, switch to Group Maker, or use the Timer
